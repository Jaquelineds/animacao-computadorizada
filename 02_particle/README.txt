# Animação Computadorizada - Jaqueline Dahmer Steffenon



02_particle - Neste projeto temos uma sistema de partículas criado a partir de scripts.
Cada partícula tem definida aleatoriamente:
- Posição (x e y são sorteados valores dentre uma margem pré estabelecida)
- Forma (esfera, cubo ou cilíndro)
- Cor (valores aleatórios para RGB)
- Trajetória (linha, espiral ou circular)

Também há evolução das partículas, cada trajetória muda a cor do objeto depois de algum tempo definido.

Foram definidos inputs de teclas do teclado também, A, B e C, que definem emitem as partículas com um padrão de posição, forma, cor e trajetória.

A morte das partículas é feita escolhendo um tempo de vida pré definido (ex.: configurado para 8 segundos).
