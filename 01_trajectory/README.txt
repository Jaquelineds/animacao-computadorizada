# Animação Computadorizada - Jaqueline Dahmer Steffenon


01_trajectory - Neste projeto temos um cenário (SampleScene) com um cilindro, onde é possível aplicar 4 tipos de trajetórias no mesmo.



Linha (arquivo Line.cs) - Trajeto linear, com movimentação de seno de x. Onde é aplicada uma função de aceleração quadrática.


Circular (arquivo Circular.cs) - Trajeto circular, aplicando seno e cosseno em x e y, respectivamente.


Espiral (arquivo Spiral.cs) - Trajeto em espiral com desaceleração linear aplicada de acordo com o aumento do tempo.


Curva Catmull (arquivo Curve.cs) - Trajeto em curva utilizando Bezier.
